PlanEasy - To-Do List Uygulaması

PlanEasy, kullanıcıların görevlerini yönetmelerini sağlayan bir masaüstü uygulamasıdır. Spring Boot backend ve JavaFX frontend kullanılarak geliştirilmiştir. Veritabanı olarak H2 (test için) veya SQLite kullanılmaktadır.

Proje Yapısı:

Backend: Spring Boot REST API

Frontend: JavaFX

Veritabanı: H2/SQLite

Özellikler:

Görev ekleme, düzenleme, silme

Hatırlatıcılar

Etiketleme ve filtreleme

Çalıştırma:

mvn clean install

mvn spring-boot:run

JavaFX arayüzü için: mvn javafx:run
